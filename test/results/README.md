# Test results

Per-round verification reports from running the daemon against TechMart. This directory **is** pushed (visible to mentors). See `claude/specs/techmart_testing.md` §8 for the report format.

Files appear here as rounds complete:

- `round_1_test_report.md` — Foundation
- `round_2_test_report.md` — Discovery (Phase 1)
- `round_3_test_report.md` — Managed sync (Phase 2)
- `round_4_test_report.md` — Comparison (Phase 3)
- `round_5_test_report.md` — BFF
- `round_6_test_report.md` — Hardening + deploy

Files prefixed with `local-` are gitignored — use that prefix for scratch reports you don't want pushed.
